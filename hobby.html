Bem-vindo à minha página sobre programação!


Estas são algumas curiosidades sobre programação:

  1. Primeiro Bug de Software: O termo "bug" para erros em software vem de um incidente real em 1947, quando programadores do Harvard Mark II encontraram uma mariposa presa em um relé do computador. Eles documentaram o evento no diário, chamando-o de "primeiro caso de bug real encontrado".
  2. O primeiro programador: Ada Lovelace, uma matemática inglesa do século XIX, é considerada a primeira programadora do mundo. Ela escreveu o primeiro algoritmo destinado a ser processado por uma máquina (a máquina analítica de Charles Babbage), mesmo antes de os computadores existirem como os conhecemos hoje.
  3. Hello, World!: O famoso programa "Hello, World!" usado para introduzir novos programadores à linguagem de programação foi popularizado por Brian Kernighan e Dennis Ritchie no livro "The C Programming Language" em 1978.
  4. COBOL e RPG ainda em uso: Muitas empresas, especialmente em setores bancário e de governo, ainda usam linguagens de programação antigas como COBOL e RPG. Elas foram escritas na década de 1960, mas continuam funcionando até hoje em sistemas legados.
  5. Especificações dos idiomas: A linguagem C é considerada uma das mais influentes, pois muitas outras linguagens, como C++, C#, Java e até mesmo Python, possuem sintaxes ou conceitos derivados dela.
  6. Tecla ESC no Vim: No editor de texto Vim, a tecla ESC é usada para alternar para o modo normal. Isso ocorre porque o criador do Vim, Bram Moolenaar, queria que o editor fosse otimizado para teclados mecânicos de computador antigos, onde a tecla ESC era mais acessível
  7. Python é nomeado em homenagem a um programa de comédia: A linguagem de programação Python recebeu esse nome em homenagem ao grupo de comédia britânico "Monty Python". O criador, Guido van Rossum, escolheu o nome porque queria que a linguagem fosse divertida e fácil de usar.
  8. Fortran ainda é relevante: Criada em 1957, a linguagem Fortran ainda é amplamente usada em ciência e engenharia, especialmente em simulações de física e modelagem de clima.
  9. Escrita a mão antes do computador: Antes do uso de terminais e IDEs, muitos programadores escreviam seus códigos em papel milimetrado para garantir que a formatação e a estruturação estivessem corretas.
  10. ASCII Art foi uma forma de arte popular nos primórdios da programação: Antes dos gráficos avançados, programadores costumavam criar "arte ASCII" usando caracteres de texto para formar imagens. Isso ainda é feito como um hobby hoje em dia.
